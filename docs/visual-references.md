# Visual References and Cues

## References
1. **Animal Crossing: New Horizons**
   - Soft pastel palette, rounded silhouettes, gentle gradients.
   - Cozy scale: small buildings, oversized foliage.
   - Warm, approachable UI with soft corners.

2. **Stardew Valley**
   - Readable zoning and paths.
   - Strong silhouette separation between props and walkable ground.
   - Clear interaction affordances on NPCs and doors.

3. **Spiritfarer**
   - Soft lighting and layered parallax depth.
   - Friendly character proportions with expressive idle motion.
   - Calm, airy color relationships.

4. **A Short Hike**
   - Simplified geometry with high contrast edges.
   - Minimalist props, clear landmarks.
   - Peaceful, low-saturation palette.

5. **Cozy Grove**
   - Warm paper-craft look with hand-drawn textures.
   - Decorative clutter without sacrificing clarity.
   - Strong use of ambient animation for liveliness.

## Key Cues to Carry Forward
- Pastel palette with gentle gradients and warm neutrals.
- Rounded silhouettes and soft corners for buildings and props.
- Clear walkable paths with subtle edge contrast.
- Ambient animation (idle bobs, subtle sway) for warmth.
- UI elements with cozy spacing and softened edges.
