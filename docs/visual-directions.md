# Visual Direction Concepts

## 1) Pastel Vector (Soft Shapes)
**Palette**
- Cream: #F7F2EC
- Blush: #F6C9C9
- Mint: #CFE9E3
- Sky: #C7DDF5
- Lavender: #D9D2F1
- Cocoa: #6B5F5A

**UI Notes**
- Rounded cards with subtle inner shadows.
- Large, friendly typography with generous padding.
- Soft gradient backgrounds and pill buttons.

## 2) Soft Pixel (Handcrafted)
**Palette**
- Warm Beige: #F2E7D8
- Peach: #F2B7A0
- Sage: #BFD8C2
- Denim: #6D8FB3
- Plum: #8A6F9E
- Charcoal: #4A4240

**UI Notes**
- Pixel frames with soft corners (2x scale).
- Dialog boxes with patterned fill.
- UI icons as simple 16x16 glyphs.

## 3) Paper-Cut (Layered Craft)
**Palette**
- Parchment: #F5EEE6
- Coral: #F4B3A2
- Moss: #A8C6B5
- Dusty Blue: #9EB5D1
- Sand: #D7C7B3
- Ink: #4A3F3A

**UI Notes**
- Layered panels with drop-shadow depth.
- Irregular edges on cards and buttons.
- Slight texture overlays to imply paper grain.
