# Village Layout Plan

## Map Sketch (Text)
```
 [Library]      [Plaza]       [Shop]
    |             |             |
 [Path] ---- [Fountain] ---- [Path]
    |             |             |
[Neighborhood]  [Player Start]  [Bar]
```

## Zones
- **Central Plaza**: fountain, benches, flower beds.
- **Library (Northwest)**: book signage, quiet nook.
- **Shop (Northeast)**: awning, crates, signboard.
- **Neighborhood (Southwest)**: small home, mailbox, garden.
- **Bar (Southeast)**: warm lights, stools, signpost.

## Pathing + Props
- Curved paths connect each zone to the plaza.
- Trees and shrubs form soft boundaries (collision).
- Decorative props: lanterns, stools, crates, flower pots.

## Player + NPC Placement
- Player start: plaza south edge.
- NPCs idle near their zone signage.
- Interaction zones in front of each building.
